# BioAgentLab

> Building AI Agents for Biology.

---

## 🚀 Vision

BioAgentLab is an open-source project focused on building AI Agents for Biology.

The long-term goal is to develop autonomous AI systems capable of:

- Literature Review
- Protein Function Analysis
- Experiment Planning
- Scientific Reasoning
- Multi-Agent Collaboration
